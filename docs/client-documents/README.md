# Client Document Pack (India)

Use these templates for every client to reduce scope, payment, and support disputes.

## Files
- `01-Proposal-Quotation.md`
- `02-Master-Service-Agreement.md`
- `03-Statement-of-Work.md`
- `04-SLA-Support-Agreement.md`
- `05-Change-Request-Form.md`
- `06-Invoice-Payment-Terms.md`
- `07-UAT-GoLive-Signoff.md`
- `08-AMC-Subscription-Addendum.md`

## Recommended Sequence
1. Share `01-Proposal-Quotation.md`.
2. Sign `02-Master-Service-Agreement.md`.
3. Attach project-specific `03-Statement-of-Work.md`.
4. Sign `04-SLA-Support-Agreement.md` (if ongoing support included).
5. Use `05-Change-Request-Form.md` for any out-of-scope work.
6. Bill with terms from `06-Invoice-Payment-Terms.md`.
7. Take acceptance using `07-UAT-GoLive-Signoff.md`.
8. Start recurring support with `08-AMC-Subscription-Addendum.md`.

## Usage Notes
- Replace all bracket placeholders like `[CLIENT_NAME]`.
- Keep one signed PDF copy per document.
- Keep one editable source copy for version tracking.
- Link document versions in your project tracker (Notion, Sheet, CRM, etc.).
- For legal enforceability in your exact case, review once with a local lawyer.
